# Aula 2 git

## aula sobre comando GIT

### git add
1. adicionar arquivos a área de "stage"
2. pode ser usado o nome do arquivo ou . ou * para adicionar todos

### git commit
1. usamos para criar um lembrente ou anotação para um ponto de mudança
2. colocamos uma mensagem
3. podemos usar o comando " git commit -a -m "mensagem"

### git push
1. para sincronizar ou subir as mudanças para o repositório remoto
2. git push -u origin "nome da branch"
brach é uma ramificação da linha do tempo do projeto

### criar novas ramificações
git branch nome_que_voce_quiser
 trocar de branch
 git checkout e o nome

 ex criar
